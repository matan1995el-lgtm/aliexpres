## 🌐 תוסף דפדפן - Browser Extension

### מה זה עושה?
התוסף מנתח אוטומטית ביקורות על מוצרים ב-AliExpress בזמן שאתה גולש!

### תכונות התוסף:
- ✅ **סריקה אוטומטית** של כל הביקורות בדף המוצר
- ✅ **ניתוח חכם** - זיהוי ביקורות חיוביות/שליליות
- ✅ **ספירת ביקורות עם תמונות**
- ✅ **מילות מפתח** - יתרונות וחסרונות
- ✅ **העתקה אוטומטית** - התוצאות מועתקות ללוח
- ✅ **כפתור צף** על דף המוצר - "נתח ביקורות ⚡"
- ✅ **התראות** - הצלחה/שגיאה

### איך להתקין?
1. פתח את התיקייה `browser-extension/`
2. **Chrome**:
   - גש ל-`chrome://extensions/`
   - הפעל "מצב מפתח" (Developer mode)
   - לחץ "טען הרחבה לא ארוזה" (Load unpacked)
   - בחר את תיקיית `browser-extension/`
3. **Firefox**:
   - גש ל-`about:debugging#/runtime/this-firefox`
   - לחץ "Load Temporary Add-on"
   - בחר את הקובץ `manifest.json`

### איך להשתמש?
1. היכנס לדף מוצר ב-AliExpress
2. לחץ על הכפתור הצף **"נתח ביקורות ⚡"**
3. המערכת תסרוק את כל הביקורות (5-10 שניות)
4. התוצאות יועתקו אוטומטית ללוח
5. חזור למערכת, הדבק בטופס "הוסף מוצר"
6. המערכת תמלא אוטומטית את כל השדות!

```javascript
נורמליזציה:
- מחיר: 200$ = 0, 0$ = 100

```

- מוצר: 35$ (75), דירוג 4.8 (96), 5000 הזמנות (50)

```javascript
if (מחיר_כולל < $75) {
    מס = 0

    מכס = מחיר_כולל × 0.12
    סה"כ = מחיר_כולל + מעמ + מכס
}
המרה_לשקלים = סה"כ × שער_דולר
```

- מחיר: 60$, משלוח: 20$ → **סה"כ: 80$**

## 📞 תמיכה ושאלות נפוצות (FAQ)

### ❓ למה המוצרים שלי נעלמו?
**תשובה**: ניקית את ה-Cache של הדפדפן. **פתרון**: גבה באופן קבוע! (הגדרות → ייצוא)

### ❓ למה המצלמה לא עובדת?
**תשובה**: לא נתת הרשאה. **פתרון**: הגדרות דפדפן → הרשאות → מצלמה → אפשר

### ❓ למה ההתראות לא מגיעות?
**תשובה**: הרשאות התראות חסומות. **פתרון**: הגדרות דפדפן → הרשאות → התראות → אפשר

### ❓ התוסף לא עובד ב-AliExpress
**תשובה**: התוסף עובד רק בדפי מוצר (*/item/*). **פתרון**: היכנס לדף מוצר ספציפי

### ❓ מחשבון המכס לא מדויק
**תשובה**: שער הדולר השתנה. **פתרון**: הגדרות → עדכן שער דולר

### ❓ איך מעבירים נתונים בין מכשירים?
**תשובה**: אין סנכרון אוטומטי. **פתרון**: 
1. מכשיר A → ייצוא JSON
2. שלח לעצמך (Email/Drive)
3. מכשיר B → ייבוא JSON

### ❓ למה אין חיפוש אוטומטי?
**תשובה**: AliExpress API דורש אישורים מסחריים. **פתרון**: חפש ידנית, המערכת תעזור בניתוח

### ❓ איך מוחקים הכל?
**תשובה**: הגדרות → "איפוס מלא של כל הנתונים" → אשר

### 🆘 עדיין יש בעיה?
1. **נקה Cache**: Ctrl+Shift+Delete
2. **רענן דף**: Ctrl+F5 (או Cmd+Shift+R על Mac)
3. **בדוק Console**: F12 → Console → חפש שגיאות אדומות
4. **בדוק גרסה**: Chrome 90+, Firefox 88+, Safari 14+
5. **נסה דפדפן אחר**: אולי הדפדפן הנוכחי לא תומך
